Input Parameters

Description of a required input of Acquisition Capabilities. This is used to describe parameters of the experiment configuration, such as direction of the antenna array beam. The description does not involve the values of the input data. The actual values, if required, shall be stored in the Data Collection as they are specific to the event of the observation and not to the applied process.
